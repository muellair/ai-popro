# learningBase_aipopro

**Owner:** Jonathan Kinkel, Felix Müller

This Docker image was created as part of the course  
**“M. Grum: Advanced AI-based Application Systems”**  
by the **Junior Chair for Business Information Science, esp. AI-based Application Systems**  
at **University of Potsdam**.

## Contents
A split of the year-wise, per-Bundesland population data in dataset [`12411-0010` from Destatis's GENESIS website](https://www-genesis.destatis.de/datenbank/online/table/12411-0010/table-toolbar).
- 80% training (`training_data.csv`), and
- 20% testing (`test_data.csv`) data.
